## Design System

Every site follows a strict set of standards:

**Typography** - No Inter, no Roboto, no system fonts. Every site gets
a unique display and body font pairing sourced from Google Fonts.

**Animation** - GSAP ScrollTrigger for all scroll animations. CSS
keyframes for micro-interactions. Nothing autoplays and loops endlessly
- everything responds to user behavior.

**Layout** - Asymmetry, overlap, and grid-breaking elements. Full-bleed
sections alternate with contained sections. Every section surprises
the eye slightly.

**Color** - Each site commits fully to one palette. No timid
half-measures. No purple gradients on white backgrounds.

**Code** - Single HTML file per site, all CSS and JS embedded. No
frameworks, no build tools, no dependencies except GSAP via CDN.
Fully mobile responsive.
